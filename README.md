# מערכת שמאות מקצועית 📋

מערכת דיגיטלית לניהול שמאויות נזק — ביטוח, פרטי ומס רכוש.

## 📁 קבצים

| קובץ | תיאור |
|------|-------|
| `index.html` | **דשבורד** — רשימת תיקים, סטטיסטיקות, ניהול |
| `form.html` | **טופס** — מערכת השמאות המלאה (9 כרטיסיות) |

## 🚀 פריסה ב-GitHub Pages

1. צור repository חדש ב-GitHub
2. העלה את שני הקבצים (`index.html`, `form.html`) לשורש הריפו
3. עבור ל-**Settings → Pages**
4. בחר **Source: Deploy from a branch** → **Branch: main** → **/ (root)**
5. לחץ **Save**
6. האתר יהיה זמין בכתובת: `https://USERNAME.github.io/REPO-NAME/`

## ✨ פיצ'רים

- **3 סוגי שמאות** — ביטוח, פרטי, מס רכוש
- **ניהול מרובה תיקים** — שמירה אוטומטית ב-localStorage
- **דשבורד** — סטטיסטיקות, פילטרים, חיפוש
- **9 כרטיסיות** — נתוני בסיס, נכס, ביטוח, אירוע, נסיבות, ממצאים, כ"כ, סיכום, תמונות
- **יצירת דוח** — PDF מוכן להגשה עם כל הנתונים
- **ייצוא/ייבוא JSON** — גיבוי וסנכרון בין מכשירים
- **מצב שטח** — ממשק מותאם לטלפון

## 💾 אחסון

כל הנתונים נשמרים **מקומית בדפדפן** (localStorage).  
אין שרת, אין מסד נתונים, אין עלות.  
לגיבוי — השתמש בכפתור "ייצוא" בדשבורד.

## 🔗 ניווט

```
index.html              ← דשבורד ראשי
  ↓ לחיצה על תיק
form.html?case=CASE_ID  ← פתיחת תיק קיים
  ↓ לחיצה על "תיק חדש"
form.html?case=CASE_ID&mode=insurance  ← תיק חדש
```
